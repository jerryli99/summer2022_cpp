First:
```
git clone https://github.com/jerryli99/summer2022_cpp.git
```
or
```
cd summer2022_cpp

git pull
```

Next:
```
cd summer2022_cpp/practices/practice3
```

Change permission:
```
chmod u+x script.bash
```

To run code, do the following: (I assume you have Docker installed)
```bash
./script.bash
```
